# AI-Based-Waste-Detection-Model
AI-based waste classification system to detect biodegradable and non-biodegradable waste using YOLOv3 and OpenCV.
Bio & Non-Bio Waste Detection using YOLOv3

This project focuses on classifying waste into biodegradable and non-biodegradable categories using a YOLOv3-based object detection model. The system processes image or video input and identifies waste type to support smart waste segregation.

Features

Detects biodegradable and non-biodegradable waste

Real-time detection using webcam or image input

Bounding box visualization with labels

Designed for smart waste management applications

Tech Stack

Python

OpenCV

YOLOv3

NumPy

Working

The system uses a trained YOLOv3 model to detect waste objects. Based on trained classes, the model classifies items into bio and non-bio categories and displays detection results with bounding boxes and labels.

Applications

Smart dustbins

Waste segregation systems

Environmental monitoring

Recycling support systems
